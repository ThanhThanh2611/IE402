# Hướng dẫn sử dụng hệ thống

Hệ thống quản lý cho thuê chung cư tích hợp bản đồ GIS 3D có 2 vai trò chính:

| Vai trò | Mô tả | Tài khoản mẫu |
|---------|-------|---------------|
| **User** | Người dùng thường — xem bản đồ, thông tin tòa nhà, căn hộ, thống kê | `user1` / `user123` |
| **Manager** | Quản lý — toàn quyền CRUD dữ liệu, quản lý hợp đồng, người dùng | `manager1` / `manager123` |

Chi tiết từng vai trò:
- [Hướng dẫn cho User](./user.md)
- [Hướng dẫn cho Manager](./manager.md)
